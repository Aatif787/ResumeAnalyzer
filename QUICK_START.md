## 💡 Quick Tips for Higher Scores

### ✅ DO
- Use strong action verbs (Developed, Managed, Led, Achieved)
- Include numbers and metrics (increased by 45%, managed $2M budget)
- Keep it concise (1-2 pages for most professionals)
- Use bullet points for easy scanning
- Include relevant keywords from job descriptions
- Add a professional summary at the top
- List 8-12 key skills
- Include contact info (email, phone, LinkedIn)

### ❌ DON'T
- Use personal pronouns (I, me, my)
- Write in paragraph form
- Include irrelevant personal information
- Use fancy fonts or complex formatting
- Exceed 2 pages (unless senior executive)
- Forget to quantify achievements
- Leave out contact information
- Use generic job descriptions

## 🔍 Common Issues & Solutions

### Issue: Low Content Score
**Solution**: Add more detailed descriptions with quantifiable achievements
- Before: "Managed team projects"
- After: "Led cross-functional team of 8 in delivering 12 projects, achieving 95% on-time completion rate"

### Issue: Low ATS Score
**Solution**: Add more relevant keywords
- Research job descriptions in your field
- Include industry-specific terms
- Add technical skills and tools
- Use both acronyms and full terms (SEO / Search Engine Optimization)

### Issue: Low Formatting Score
**Solution**: Improve structure and organization
- Use clear section headers (EXPERIENCE, EDUCATION, SKILLS)
- Convert paragraphs to bullet points
- Ensure consistent formatting throughout
- Add complete contact information

### Issue: No Metrics Found
**Solution**: Quantify your achievements
- Revenue/Sales: "Increased sales by 35%"
- Efficiency: "Reduced processing time by 40%"
- Scale: "Managed team of 15 people"
- Impact: "Saved company $500K annually"

## 🎨 Formatting Best Practices

### Font
- **Recommended**: Arial, Calibri, Georgia, Times New Roman
- **Size**: 10-12pt for body, 14-16pt for name
- **Avoid**: Comic Sans, decorative fonts

### Margins
- **Standard**: 0.5-1 inch on all sides
- **Consistent**: Same on all pages

### Spacing
- **Line Spacing**: 1.0-1.15
- **Section Spacing**: 1-2 blank lines between sections

### Colors
- **Safest**: Black text on white background
- **Acceptable**: Dark gray text, subtle accent colors for headers
- **Avoid**: Bright colors, multiple colors

## 🤖 ATS (Applicant Tracking System) Tips

### What is ATS?
ATS software scans and ranks resumes before humans see them. To pass ATS screening:

### ✅ ATS-Friendly Practices
- Use standard section headers
- Avoid headers/footers
- Don't use tables or text boxes
- Save as .docx or .pdf
- Include keywords from job description
- Use standard fonts
- Spell out acronyms first time

### ❌ ATS Killers
- Images or graphics
- Complex formatting
- Columns
- Headers/footers
- Special characters
- Unusual section names

## 📞 Troubleshooting

### Resume Won't Upload
- Check file format (PDF, DOCX, TXT only)
- Ensure file size is under 10MB
- Try a different browser
- Check if JavaScript is enabled

### Analysis Taking Too Long
- Refresh the page and try again
- Check internet connection (for loading CDN libraries)
- Try with a smaller file

### PDF Not Parsing Correctly
- Complex formatting may not extract perfectly
- Try saving as plain text PDF
- Convert to .docx format
- Check if PDF has text layer (not scanned image)
